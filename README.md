# Hadia Gold Inventory Management System - Frontend

## Overview

The **Hadia Gold Inventory Management System Frontend** is a modern, responsive React application developed for **HadiaGold Group Ltd.**, headquartered in **Almaty, Kazakhstan**.

The application provides an intuitive interface for managing inventory, purchases, sales, suppliers, customers, companies, dealers, and financial reporting. It also offers multilingual support, secure authentication, and a responsive design for desktop and mobile devices.

# Multi-language Support

The application supports multiple languages:

* English
* Persian (Dari)
* Russian

Users can seamlessly switch between languages to improve accessibility and user experience.

---

# Authentication

The frontend integrates with the backend authentication system and provides:

* Secure Login
* JWT Authentication
* Protected Routes
* Role-Based Access
* Email OTP Verification
* Password Recovery via Email

Administrators have access to a complete management dashboard where they can manage:

* Products
